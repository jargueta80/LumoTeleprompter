## Required Permissions Explanation

### Android
- **INTERNET:** Required for local WebSocket communication between devices
- **ACCESS_WIFI_STATE:** Required to display local IP address for remote connection
- **ACCESS_NETWORK_STATE:** Required to check network connectivity status

### iOS
- **Local Network:** Required to enable remote control functionality between devices on the same Wi-Fi network
